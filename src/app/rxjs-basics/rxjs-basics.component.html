RxJS Basics

Reactive Extensions for JavaScript or short RxJS is an Functional Reactive Programming Framework for working with ansynchrounus and event based data streams with pure functions. Behind ReactiveX is the Observable Pattern, which combines the Observer and Iterrator Pattern. With this combination Observers get notified about changes, completion and erros.

Observer Pattern

The Observer Pattern consists of two players: Subject and Observer. The Subject holds a list of observers, which can be added, removed and notified about changes. The Observer has one update method, which is called by the Subject.

Iterrator Pattern

The Iterrator Pattern is similar to foreach loops where an iterator goes over the items of a list.

Coe Concepts

Observables are classes representing a Stream of data. An Observer is a class for emitting data inside an Observable. With Operators that are immutable methods Streams can be manipulated. 

      let stream$ = Observable.create(observer => observer.next(5));
      stream$ = stream$.pipe(map(x => x as number + 8));
      stream$.subscribe(data => console.log(data));

List of some Operators

Creation Operators

  • of() - Creates nearly from everything an observable
  • fromEvent() - Creates an observable from JavaScript events

Transforming Operators

  • map() - transform the items emitted by an Observable by applying a function to each item

Filter Operators

  • debounce() - only emit an item from an Observable if a particular timespan has passed without it emitting another item
  • filter() - emit only those items from an Observable that pass a predicate test
